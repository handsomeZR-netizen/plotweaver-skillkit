# Style Decision Rules

- Use serif typography. Prefer Times New Roman for Latin text and STIX for math.
- Prefer restrained palettes instead of Matplotlib or seaborn defaults.
- Keep grids light so they do not overpower the data.
- Place legends outside dense data regions when possible.
- Export `png`, `pdf`, and `svg` at high resolution.
- Reuse theme, spacing, and color philosophy before copying implementation details.
- If code extraction confidence is low, reuse the visual language without copying the original logic verbatim.
