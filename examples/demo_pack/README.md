# Demo Pack

This folder is a lightweight public example of the artifact contract produced by `wechat-plotkit`.

Included files:

- `master_style_index.json`
- `template_registry.json`
- `reuse_playbook.md`
- `validation_manifest.json`
- `style_profile_radar.json`
- `article_radar.json`

Purpose:

- show the exact JSON and Markdown shapes consumed by the reuse workflow
- keep the public repository compact
- avoid publishing the full raw local scrape archive

Use this pack for README examples, template selection checks, and starter-script generation.
